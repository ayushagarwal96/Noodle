What is Noodle?

Noodle is an Intra-Campus Online Book Portal where students can buy and sell books with any hassle.


How to use Noodle?

Noodle is a very intuitive, user-friendly and easy-to-use Web Application. Just register, sign-in, add a book(s) you want to sell and/or use the search bar to search for a book you want to buy that might be for sale.
